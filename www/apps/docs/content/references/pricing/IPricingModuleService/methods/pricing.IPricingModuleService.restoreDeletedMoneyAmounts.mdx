# restoreDeletedMoneyAmounts - Pricing Module Reference

This documentation provides a reference to the `restoreDeletedMoneyAmounts` method. This belongs to the Pricing Module.

This method restores soft deleted money amounts by their IDs.

## Example

```ts
import {
  initialize as initializePricingModule,
} from "@medusajs/pricing"

async function softDeleteMoneyAmounts (moneyAmountIds: string[]) {
  const pricingService = await initializePricingModule()

  await pricingService.softDeleteMoneyAmounts(
    moneyAmountIds
  )
}
```
